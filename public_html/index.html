Meta-Jerseyclose

Meta-Jersey provides a tool for journalists that can aid them in understanding what is causing shifts in the New Jersey census. By using a wide range of data, which is visualized on an interactive heat map, users will be able to chose what data is displayed on the map, allowing them to create a story based on the data that is being visualized.

This project was built during the Hack Jersey 2.0, with a team made up of both coders and journalists, where we won the "Best Overall" award.

MSU Illuminatemore_vert
MSU Illuminateclose

MSU Illuminate was an idea that I envisioned while trying to create a project for the MSU Computer Science club to spend the next academic year working on. The purpose of the project was to create an interactive infinity light mirror, which students could sent a tweet to, defining what color they want the lights inside the mirror to be.

To make this project possible, the MSU Computer Science club collaborated with the MSU Industrial Design club, allowing the Computer Science students to focus primarily on the electronic compnents and software, leaving the Industrial Design students to design and build the structure.

Before I graduated, I was able to assist the Computer Science students in completing all the hardware (consisting of a Raspberry Pi and lots of RGB LED lights), and the web interface that would be displayed on a screen in the middle of the mirror. Unfortunately, I had to hand off the reins of the project, since the Industrial Design club hadn't quite finished the structure for the electrical compnents at the time.

I will be writing an indepth blog post about my experiences throughout this project very soon.

Whitespace Database Debug Toolmore_vert
Whitespace Database Debug Toolclose

During the Summer between my Sophomore and Junior year of college, I was a student intern for Applied Communication Sciences (formally Telcordia Technologies), and I was assigned to work on a HAAT(Height Above Average Terrain) tool for iconnectiv's White Space Database.

The purpose of this tool is to retrieve data from the White Space Database about the height above the average terrain for specific areas, and visualize this data on a map. The results from this tool can then be used to determine the best location for new radio towers, allowing the user to ensure optimal signal coverage for a new tower. Additionally, this tool can be used for troubleshooting existing towers, to learn why there may be poor signal coverage in a specific area.
